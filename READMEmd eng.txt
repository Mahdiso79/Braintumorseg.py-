This program is created using PyQt5 for the user interface, OpenCV for image processing, TensorFlow for deep learning, and Matplotlib for visualization.

The user is prompted to select an image of the brain. Then, they can perform various operations using different buttons, including:
- Converting the image from color to grayscale
- Applying bilateral filter
- Applying median filter
- Applying Gaussian filter
- Applying thresholding
- Displaying a 3D image
